Drop your video + poster files in this folder using these exact names
and they'll show up automatically (index.html already points to them):

  video-osmo-pocket-3.mp4       poster-osmo-pocket-3.jpg
  video-neutrogena-collab.mp4   poster-neutrogena-collab.jpg
  video-retinal-shot.mp4        poster-retinal-shot.jpg

Notes:
- Vertical (9:16) video works best — it fills the phone frame edge to edge.
- Until a video is added, that card just shows a soft gradient placeholder
  with a play button, so the site never looks broken.
- Poster images are optional; if omitted, the first frame area just shows
  the gradient until the video is played.
- Recommended: keep each file under ~15MB (trim/compress) so the page
  loads quickly, especially on mobile data.
